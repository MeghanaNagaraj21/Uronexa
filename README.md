# Uronexa

Uronexa is an AI-assisted system designed to improve interpretation of UTI dipstick test results by reducing color ambiguity and improving diagnostic clarity.

## Problem
UTI dipstick tests often produce similar color variations that are difficult to interpret accurately.

## Solution
Uronexa uses image analysis and AI-assisted interpretation to provide clearer diagnostic outputs.

## Features
- Dipstick color interpretation
- AI-assisted prediction
- User-friendly analysis
- Reduced diagnostic confusion

## Tech Stack
- Python
- Machine Learning
- Image Processing
- Computer Vision
